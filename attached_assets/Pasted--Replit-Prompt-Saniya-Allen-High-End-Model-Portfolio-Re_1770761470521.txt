# Replit Prompt: Saniya Allen ‚Äî High-End Model Portfolio Redesign

## Project Overview

Build a **static site** (HTML/CSS/JS ‚Äî no frameworks, no build tools) deployable to **Netlify** for model **Saniya Allen**. This is a full redesign of her current portfolio at [saniyallen.com](https://www.saniyallen.com/). The current site is a basic Format.com template with a flat image grid. The redesign should feel like a **luxury fashion editorial website** ‚Äî think Vogue digital, high-end agency books, or Awwwards-winning model portfolios.

---

## Tech Stack (Keep It Simple & Static)

- **Pure HTML5, CSS3, JavaScript** ‚Äî no React, no Vue, no build step
- **GSAP 3 + ScrollTrigger** (CDN) ‚Äî for all scroll animations, parallax, pinning
- **Lenis** (CDN) ‚Äî for buttery smooth scrolling
- **CSS Custom Properties** for theming
- **Google Fonts** ‚Äî use a high-fashion serif + clean sans-serif pairing
- Deploy-ready for **Netlify** (just an `index.html` + assets folder)

## Section-by-Section Design Specs

### 1. HERO ‚Äî Cinematic Full-Screen Intro
- **Full-viewport** landing with a single striking hero image of Saniya (or a looping short video clip if available)
- Her name `SANIYA ALLEN` in **oversized, elegant serif typography** (like Playfair Display or Cormorant Garamond) ‚Äî minimum 10vw font size
- Text should **split-animate in** letter by letter using GSAP SplitText-style effect (manually split spans)
- Subtitle text: `MODEL ¬∑ CREATIVE ¬∑ MUSE` in a small tracked-out sans-serif beneath
- **Parallax**: hero image moves at 0.5x scroll speed, text moves at 1.2x ‚Äî creates depth
- Smooth **clip-path reveal** animation on load ‚Äî image reveals from center outward like a curtain opening
- Navigation: minimal floating nav (transparent, appears on scroll) with links: `Features` | `UGC` | `Inquiries`
- **Mouse-follow subtle glow effect** ‚Äî a soft radial gradient follows the cursor over the hero

### 2. ABOUT ‚Äî Elegant Bio Reveal
- Two-column layout: large portrait image on left (60% width), text on right
- Image has a **parallax float** ‚Äî moves slightly slower than scroll
- Text fades in with a **staggered line-by-line reveal** triggered by ScrollTrigger
- Typography: elegant serif for name, clean sans-serif for body
- Include a subtle **horizontal line animation** that draws itself across the section
- Bio text example: *"Nashville-based model with a passion for editorial storytelling, brand campaigns, and creative direction."*
- Keep it short ‚Äî 3-4 sentences max. Let the visuals do the talking.

### 3. FEATURES GALLERY ‚Äî Magazine-Style Scroll Experience
- **Horizontal scroll section** pinned to viewport ‚Äî user scrolls vertically but images move horizontally
- Each image is large (70-80vh height), with a slight **rotation on scroll** (¬±2deg tilt as they enter/exit)
- Images should have a **film-grain overlay** (CSS pseudo-element with noise texture)
- On hover: images scale up 1.05x with a smooth ease, and a **caption slides up** from the bottom (shoot name, publication, year)
- Between images, use **varied spacing** ‚Äî not a rigid grid. Some images overlap slightly. Editorial chaos.
- Section header: `FEATURES` in huge outlined/stroke text that scrolls at a different speed (parallax text)
- ScrollTrigger scrub animation ‚Äî images are tied to scroll position, not time

### 4. UGC GALLERY ‚Äî Masonry / Staggered Grid
- Masonry-style grid with **staggered fade-in** animations (each image enters with a slight delay and Y-offset)
- Mix of portrait and landscape orientations
- On hover: **grayscale-to-color transition** (images start slightly desaturated, full color on hover)
- **Infinite scroll feel** ‚Äî images populate as user scrolls down
- Lightbox on click: full-screen image view with blur backdrop and smooth scale-up animation
- Section header: `UGC` with the same oversized outlined text treatment as Features

### 5. ROTATING 3D GALLERY ‚Äî The Showstopper ‚ú®
- **3D CSS carousel** ‚Äî images arranged in a circular ring using `transform: rotateY()` and `translateZ()`
- Auto-rotates slowly, but user can **drag to spin** (touch + mouse)
- Each image card has a subtle **reflection effect** underneath (CSS pseudo-element with opacity gradient)
- On hover over a card: it **pops forward** (translateZ increases), slight glow/shadow
- The whole carousel sits on a **dark background section** with a subtle radial gradient
- Smooth GSAP-powered rotation with inertia (keeps spinning a bit after drag release)
- Mobile: falls back to a **swipeable horizontal carousel** with snap points

### 6. STATS / HIGHLIGHTS TICKER
- Horizontal auto-scrolling ticker strip (like a stock ticker or news crawler)
- Content: `‚ú¶ VOGUE ‚ú¶ ELLE ‚ú¶ 50K+ FOLLOWERS ‚ú¶ NASHVILLE ‚ú¶ AVAILABLE WORLDWIDE ‚ú¶` (repeat infinitely)
- Pure CSS animation (`@keyframes translateX` loop) ‚Äî lightweight, no JS needed
- Sits between sections as a visual break
- **Two strips** moving in opposite directions for a dynamic feel
- Monospaced or tight sans-serif font, all caps, tracked out

### 7. CONTACT / INQUIRIES
- Clean, minimal form: Name, Email, Subject (dropdown: Booking / Collaboration / General), Message
- **Floating label inputs** ‚Äî labels animate up when focused
- Submit button with a **magnetic hover effect** (button subtly follows cursor when near)
- Background: subtle gradient or a blurred version of a portfolio image
- Include social links: Instagram icon, Email icon ‚Äî minimal, elegant
- Form submits to **Netlify Forms** (just add `netlify` attribute to form tag ‚Äî zero backend needed)
- Success state: the form fades out and a `Thank you ‚ú¶` message fades in

### 8. FOOTER
- Minimal: `¬© 2025 Saniya Allen. All Rights Reserved.`
- Small social icons row
- **Scroll-to-top** smooth scroll on logo click
- The entire footer section has a **reveal animation** ‚Äî content slides up as it enters viewport

## Global Design System

### Color Palette
```css
:root {
  --bg-primary: #0a0a0a;         /* Rich black */
  --bg-secondary: #141414;       /* Soft black for alternating sections */
  --text-primary: #f5f0eb;       /* Warm off-white / cream */
  --text-secondary: #a09890;     /* Muted warm gray */
  --accent: #c9a96e;             /* Warm gold ‚Äî for highlights, links, hover states */
  --accent-hover: #e0c68a;       /* Lighter gold for hover */
}
```

### Typography
```
Headings: "Cormorant Garamond", serif ‚Äî elegant, high-fashion
Body: "Inter" or "Outfit", sans-serif ‚Äî clean, modern
Accent/Ticker: "Space Grotesk" or "JetBrains Mono" ‚Äî for the ticker strip
```

### Global Animations & Micro-Interactions
- **Custom cursor**: Replace default cursor with a small circle that scales up on hoverable elements
- **Smooth scroll**: Use Lenis for native-feeling smooth scroll across the whole page
- **Page load sequence**: Black screen ‚Üí logo fade-in ‚Üí clip-path reveal of hero ‚Üí nav slides down
- **All scroll animations**: Use GSAP ScrollTrigger with `scrub: true` for scroll-linked motion
- **Image loading**: Use `loading="lazy"` on all images, add a subtle blur-to-sharp placeholder transition
- **Page transitions**: Since it's single-page, use smooth scrollTo for nav links with GSAP ScrollTo plugin

## Responsive Breakpoints

```css
/* Mobile First */
@media (min-width: 768px)  { /* Tablet */ }
@media (min-width: 1024px) { /* Desktop */ }
@media (min-width: 1440px) { /* Large Desktop */ }
```

- Mobile: Stack everything vertically, horizontal scroll becomes vertical swipe gallery, 3D carousel becomes horizontal swipe
- Touch: All hover effects should have tap equivalents
- Performance: Reduce/disable parallax on mobile (`ScrollTrigger.matchMedia`)

## Netlify Deployment Notes

- Add `netlify` attribute to the contact form: `<form name="contact" method="POST" data-netlify="true">`
- Add a hidden input for bot protection: `<input type="hidden" name="form-name" value="contact">`
- Optional `netlify.toml`:
```toml
[build]
  publish = "/"

[[headers]]
  for = "/*"
  [headers.values]
    X-Frame-Options = "DENY"
    X-Content-Type-Options = "nosniff"
    Cache-Control = "public, max-age=31536000"
```
